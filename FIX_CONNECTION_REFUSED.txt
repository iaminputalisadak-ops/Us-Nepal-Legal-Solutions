âš ï¸  IF APACHE WON'T START:

Problem: Port 80 is already in use
Solution:
1. In XAMPP, click "Config" next to Apache
2. Select "httpd.conf"
3. Find "Listen 80" and change to "Listen 8080"
4. Save and close
5. Start Apache again
6. Update backend URL to: http://localhost:8080/backend/

Problem: Permission denied
Solution:
1. Right-click XAMPP Control Panel
2. Select "Run as Administrator"
3. Try starting Apache again

Problem: Apache starts then stops immediately
Solution:
1. Click "Logs" button next to Apache in XAMPP
2. Check error.log for specific error
3. Common fix: Change port from 80 to 8080 (see above)
